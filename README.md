# leaflet-global-switches



## Description
Setting [Global Switches](https://leafletjs.com/reference.html#global-switches) for [Leaflet](https://leafletjs.com/)

## Installation
### bower
`bower install https://github.com/FCOO/leaflet-global-switches.git --save`
